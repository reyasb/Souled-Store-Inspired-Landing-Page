# Soul Store Inspired Landing Page

A simple and responsive landing page inspired by the **Souled Store** brand, built using only **HTML** and **CSS**. This project was created to improve my front-end development skills by recreating a modern e-commerce website layout.

> **Note:** This project is for learning purposes only and is inspired by the design of the Souled Store website. It is not an official Soul Store website.

---

## 🚀 Features

- Modern landing page design
- Responsive navigation bar
- Hero section
- Product showcase
- Clean and organized layout
- Hover effects
- Pure HTML & CSS (No JavaScript)

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

## 🎯 Purpose

This is one of my first front-end projects. Through this project, I learned:

- HTML page structure
- CSS styling
- Flexbox
- Responsive layouts
- Positioning
- Hover effects
- Basic UI design
